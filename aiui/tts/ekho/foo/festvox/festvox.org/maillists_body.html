Mailing Lists
There are two main mailing lists, one for discussing Festival in general and another more specific on Voice Building using the FestVox tools. If you subscribe to the FestVox list you should probably also subscribe to the Festival list too, but if you are not building new voice the Festival list should be adequate.
  • The main mailing list for discussing issues related to the University of Edinburgh's Festival Speech Synthesis System is festival-talk@festvox.org

    To join the list send a message to majordomo@festvox.org with the following line in the body of the message 

        subscribe festival-talk

    The mailing list is also archived

    • festival-talk archive (to be added ...)
    This list was hosted at Edinburgh up until July 2004.
  • The primary mailing list for discussion of building voices is festvox-talk@festvox.org.

    To join the list send a message to majordomo@festvox.org with the following line in the body of the message 

        subscribe festvox-talk

    The mailing list is also archived

    • festival and festvox discussion list is archived at https://lists.berlios.de/pipermail/festlang-talk/
