CSTR UK RAB Diphone
This databases consists of a set of nonsense words containing all phone-phone transitions for UK English. This database is free for any use (see licence for details). This database includes, waveforms, laryngograph (EGG) files, hand corrected labels, extracted pitchmarks, and various support files. It is released both as an example and in the hope you can make the festvox_rab voice sound better.
